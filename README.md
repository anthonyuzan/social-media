# social-media

Stack: React, Firebase, Redux, Express and Material-UI.

## What's done?:

Back: 
- Read posts
- Create new posts
- Add comments to a post
- Like and unlike posts
- Notifications system 
- Delete posts
- Authentification security
- Signup 
- Login
- Change profile image

Front:
- Main pages
- Routes
- Form to login
- Form to sign up
